(xiii) Признаки сходимости рядов
19.04.2016

Признак Д'Аламбера:

\[ \lim_{n \to \infty }{\frac{a_{n+1}}{a_n}} = L\]

Признак Коши:

\[ \lim_{n \to \infty }{\sqrt[n]{a_n}} = L\]

при $L < 1$ сходится, $L > 1$ расходится, $L = 1$ — неизвестно.

Признак Раабе (ряд строго положителен):

\[ \lim_{n \to \infty }{n(\frac{a_n}{a_{n+1}} - 1)} = p\]

при $p > 1$ сходится, $p < 1$ расходится, $p = 1$ — неизвестно.

Если ряд $\sum_{n=1}^{\infty}{a_n}, a_n > 0$ с монтонно убывающими членами сходится, то:

\[ \lim_{n \to \infty}{na_n} = 0 \]

верно обратное $\lim{na_n} \neq 0$ — расходится.

Интегральный признак Коши. Если функция $f$ неотрицательная при $x > 0$ и не возрастает, то ряд $\sum_{n=1}^{\infty}{f(n)}$ сходится или расходится одновременно с несобственным интегралом:

\[ \int_{1}^{\infty}{f(x) dx}\]

Из абсолютной сходимости ряда следует его сходимость.

Признак Лейбница. Если $a_n = (-1)^n b_n, \; b_n \geq 0$, и последовательность $(b_n)$, начиная с некоторого номера $n_0$, монотонно стремится к нулю, то ряд $\sum_{n=1}^{\infty}{a_n}$ сходится.

Признак Абеля. Ряд $\sum_{n=1}^{\infty}{a_n b_n}$ сходится, если сходится ряд $\sum_{n=1}^{\infty}{a_n}$ и последовательность $(b_n)$ есть монотонная и ограниченная.

Принак Дирихле. Ряд $\sum_{n=1}^{\infty}{a_n b_n}$ сходится, если последовательность $(b_n)$, начиная с некоторого номера $n_0$, монотонно стремится к нулю, а последовательность частичных сумм ряда $\sum_{n=1}^{\infty}{a_n}$ ограничена.

(xii) Перестановки и свойства
18.04.2016

Перестановка (подстановка) $n$ чисел: $(1, 2, ..., n-1, n)$

Число различных перестановок: $n!$

Число различных циклов из $k$ элементов: $(k-1)!$

Число коммутирующих перестановок с данной $\sigma$ ($\sigma \pi = \pi \sigma$):

\[ \prod_{i = 1}^{k}{n_i^{m_i} \cdot m_i ! } \]

где $n_i$ — уникальная длина цикла, $m_i$ — кол-во циклов с длиной $n_i$ в перестановке $\sigma$.

(xi) О числах
02.04.2016

Разложение натурального $n$ — упорядоченкая сумма натуральных слагаемых. В общем случае существуюет $2^{(n-1)}$ разложений (композиций) числа $n$, из которых в точности $C_{n-1}^{k-1}$ имеют длину $k$. Разрешая нулевые слагаемые: $C_{n+k-1}^{k-1}$ разложений, произвольной длины — бесконечно много.

Малая теорема Ферма: если $p$ — простое числа и $a$ — целое число, не делящееся на $p$, то $a^{p-1} - 1$ делится на $p$, то есть $a^{p-1} = 1 \; (mod \; p)$.

(x) Неравенство о средних
07.02.2016

Для любых неотрицательных $x_1, x_2, ..., x_n$ верно неравенство:

\[ \bar{x}_{kvadr} \geq \bar{x}_{arithm} \geq \bar{x}_{geom} \geq \bar{x}_{garmon}\] \[ \bar{x}_{kvadr} = \sqrt{\frac{1}{n}\sum_{i=1}^{n}{x_i^2}} \] \[ \bar{x}_{arithm} = \frac{1}{n}\sum_{i=1}^{n}{x_i} \] \[ \bar{x}_{geom} = \sqrt[n]{\prod_{i=1}^{n}{x_i}} \] \[ \bar{x}_{garmon} = \frac{n}{\sum_{i=1}^{n}{\frac{1}{x_i}}} \]

(ix) Подобные матрицы. Жорданова форма
03.02.2016

Матрица $\mathbf{A}$ подобна $\mathbf{B}$, если существует невырожденная $\mathbf{C}$ такая, что: \[\mathbf{A} = \mathbf{C}^{-1}\mathbf{B}\mathbf{C}\]

У подобных матриц совпадает: определитель, ранг, след, собственные значения (собственные векторы не обязательно совпадают). В вышеприведенном равенстве, $\mathbf{C}$ выступает в качестве матрицы перехода от одного базиса к другому.

Жорданова клетка: \[ \mathbf{J}_{\lambda} = \left( \begin{array}{ccc} \lambda & 1 & 0 \\ 0 & \lambda & 1 \\ 0 & 0 & \lambda \end{array} \right)\]

Для любой матрицы $\mathbf{A}$ существует невырожденная $\mathbf{C}$ такая, что: \[\mathbf{A} = \mathbf{C}^{-1}\mathbf{J}\mathbf{C}\]

$\mathbf{J}$ — матрица, на диагонали которой находятся различные жордановы блоки $\mathbf{J}_{\lambda_i}$. $\lambda_i$ — собственные значения $\mathbf{A}$. Если все жордановы клетки имеют размерность $1$, то матрица диагональная. Для эрмитовой матрицы ($\mathbf{A} = \mathbf{A}^{*}$, где $\mathbf{A}^{*}$ — комплексно-сопряженная и транспонированная, частный случай — действительная симметричная матрица) все жордановы клетки имеют размерность $1$.

Жордановы формы двух подобных матриц совпадают с точностью до порядка клеток. Матрица $\mathbf{A}$ подобна $\mathbf{A}^T$.

(viii) Дополнительные свойства матриц
24.01.2016

Несколько дополнительных свойств матриц:

  • Если $\mathbf{AB} = \mathbf{BA}$, то матрицы $\mathbf{A}$ и $\mathbf{B}$ называются коммутирующими между собой.
  • $\mathbf{A}(\mathbf{B} + \mathbf{C}) = \mathbf{AB} + \mathbf{BC}$.
  • $tr(\mathbf{AB}) = tr(\mathbf{BA})$.
  • $(\mathbf{AB})^T = \mathbf{B}^T\mathbf{A}^T$.
  • $(\mathbf{AB})^{-1} = \mathbf{B}^{-1}\mathbf{A}^{-1}$. Доказательство: \[\mathbf{AB} = \mathbf{C}\] \[\mathbf{B}^{-1}\mathbf{A}^{-1}\mathbf{AB}\mathbf{C}^{-1} = \mathbf{B}^{-1}\mathbf{A}^{-1}\mathbf{C}\mathbf{C}^{-1}\] \[\mathbf{B}^{-1}\mathbf{E}\mathbf{B}\mathbf{C}^{-1} = \mathbf{B}^{-1}\mathbf{A}^{-1}\mathbf{E}\] \[\mathbf{B}^{-1}\mathbf{B}\mathbf{C}^{-1} = \mathbf{B}^{-1}\mathbf{A}^{-1} \; \Rightarrow \; \mathbf{C}^{-1} = \mathbf{B}^{-1}\mathbf{A}^{-1}\]
  • $(\mathbf{A}^{-1})^T = (\mathbf{A}^T)^{-1}$
  • $(\mathbf{A} + \mathbf{B})^T = \mathbf{A}^T + \mathbf{B}^T$.
  • $tr(\mathbf{A}) = tr(\mathbf{A}^T)$
  • $\ln{det(\mathbf{A})} = tr(\ln{\mathbf{A}})$ (разобраться с матричной экспонентой!)
  • $tr(\mathbf{A}) = \sum_{i}^{n}{\lambda_i}$
  • $tr(\mathbf{A}^m) = \sum_{i}^{n}{\lambda_i^m}$

(vii) Свойства определителя $|\mathbf{A}|$
28.11.2015

Разные виды записи и подсчет через миноры при разложении по $i$-ой строке:

\[det(\mathbf{A}) \equiv \Delta_A \equiv |\mathbf{A}| = \sum_{j=1}^{n}{(-1)^{i+j}a_{ij}\bar{M}_i^j}\]

Свойства:

  • При добавлении к любой строке (столбцу) линейной комбинации других строк (столбцов) определтель не меняется.
  • Если две и более строки (столбца) линейно зависимы (в частности равны), то $|\mathbf{A}| = 0$.
  • Если две строки (столбца) поменять местами, определитель сменит знак $|\mathbf{A}'| = -|\mathbf{A}|$.
  • Общий множитель строки (столбца) можно вынести за знак детерминанта. В частности $|k\mathbf{A}| = k^n|\mathbf{A}|$.
  • Если в $|\mathbf{A}|$ есть нулевая строка (столбец), то $|\mathbf{A}| = 0$.
  • $|\mathbf{AB}| = |\mathbf{A}||\mathbf{B}|$.
  • $|\mathbf{A}| = \prod_{i = 1}^n{\lambda_i}$, где $\lambda_i$ — $i$-ое собственное значение матрицы $\mathbf{A}$.
  • Свойство линейности: Если элементы какой-либо строки представлены в виде суммы двух слагаемых, то определитель равен сумме двух определителей, в первом из которых элементы отмеченной строки равны первым слагаемым, во втором — вторым.
  • $|\mathbf{A}| = \prod_{i = 1}^{n}{a_{ii}}$, если $\mathbf{A}$ треугольная матрица (верхне- или нижнетреугольная).
  • $|\mathbf{A}^T| = |\mathbf{A}|$.
  • Чтобы матрица $\mathbf{A}$ имела обратную $\mathbf{A}^{-1}$, необходимо чтобы $|\mathbf{A}| \neq 0$.

(vi) Поправки на множественное сравнение
06.11.2015

Проведена серия из $n$ статистических тестов. Отсортируем их $p$ по возрастанию: $p_1 \leq p_2 \leq ... \leq p_n$. Рассмотрим несколько поправок на множественное сравнение (от жестких к мягким):

Поправка Бонферрони:

\[\widetilde{p}_i = {p_i} \cdot {n}\]

Поправка методом Холма:

\[\widetilde{p}_i = {p_i} \cdot {(n - i + 1)}\]

Поправка Бенджамини-Хохберга:

\[\widetilde{p}_i = p_i \cdot \frac{n}{i}\]

В поправке Б-Х есть дополнительное условие $\widetilde{p}_i = min(p_i \cdot \frac{n}{i}, \widetilde{p}_{i-1})$

Во всех корректировках $\widetilde{p}_i = min(1, ...)$

(v) Математическое ожидание. Дисперсия
25.10.2015

Математическое ожидание непрерывной случайной величины:

\[M[X] = \int_{-\infty}^{\infty}{x f_x(x) dx}\]

Дисперсия ($D[X]$ или $Var(X)$):

\[D[X] = M[(X - M[X])^2] = \sigma^2\]

$\sigma$ — стандартное или среднеквадратичное отклонение. Удобная запись:

\[D[X] = M[X^2] - (M[X])^2 \]

Неравенство Чебышева:

\[P(|X - \mu| \geq a) \leq \frac{\sigma^2}{a^2}\]

Для неотрицательной целочисленной случайной величины:

\[M[X] = \sum_{i = 1}^{\infty}{P(X \geq i)} \]

Для неотрицательной непрерывной случайной величины:

\[M[X] = \int_{0}^{\infty}{P(X \geq x) dx} \]

(iv) Несколько свойств $C_n^k$
25.10.2015

Два вида записи:

\[C_n^k \equiv \binom{n}{k} = \frac{n!}{k! (n-k)!}\]

Бином Ньютона (для всех $k > n$ : $C_n^k = 0$):

\[(1+x)^n = \sum_{k = 0}^{\infty}{C_n^k x^k}\]

Сумма соседей:

\[C_n^k + C_n^{k+1} = C_{n+1}^{k}\]

Разность:

\[C_{n}^{k} - C_{n-1}^{k-1} = C_{n-1}^{k}\]

Из задачи (ШАД):

\[\sum_{t = 0}^m{C_{s+t}^{t}} = C_{s + m + 1}^{m}\]

Сумма:

\[\sum_{m = 0}^n{C_{m}^{k}} = C_{n+1}^{k+1}\]

(iii) Интегрирование по частям
25.10.2015

Формула интегрирования по частям:

\[\int{u dv} = uv - \int{v du}\]

Удобная запись:

\[\int{uv' dx} = uv - \int{u'v dx}\]

Стоит помнить, что в неопределённом виде равенство выполняется с точностью до константы. Ошибочный парадокс:

\[\int{\frac{dx}{x}} = \frac{1}{x}\cdot{x} - \int{\frac{-1}{x^2} \cdot xdx} = 1 + \int{\frac{dx}{x}} \; \Rightarrow \; 0 = 1\]

Один из примеров (ШАД):

\[\Phi = \int_a^b{\varphi(x)\varphi'(x)dx} = \varphi^2(x) \bigg|_a^b - \int_a^b{\varphi(x)\varphi'(x) dx} = \varphi^2(x) \bigg|_a^b - \Phi\] \[\Phi = \frac{1}{2} \cdot \varphi^2(x) \bigg|_a^b\]

(ii) Прогрессии
24.10.2015

Арифметическая прогрессия:

\[a_n = a_1 + (n-1)d\] \[S = \frac{a_1 + a_n}{2} \cdot n\]

Геометрическая прогрессия:

\[b_n = b_1 q^{n-1}\] \[|b_n| = \sqrt{b_{n-1}b_{n+1}}\] \[S = \frac{b_1(1-q^n)}{1-q}\]

Бесконечно убывающая геометрическая прогрессия ($|q| < 1$):

\[S_n \to \frac{b_1}{1 - q}, n \to \infty\]

(i) Гармонические ряды
24.10.2015

Гармонический ряд:

\[S = 1 + \frac{1}{2} + \frac{1}{3} + ... = \sum_{k = 1}^{\infty}{\frac{1}{k}}\]

Ряд расходится. Аналогичный знакопеременный ряд сходится:

\[S = 1 - \frac{1}{2} + \frac{1}{3} - ... = \sum_{k = 1}^{\infty}{(-1)^{k+1} \frac{1}{k}} = \ln 2\]

Обобщённый гармонический ряд — ряд Дирихле:

\[S = 1 + \frac{1}{2^{\alpha}} + \frac{1}{3^{\alpha}} + ... = \sum_{k = 1}^{\infty}{\frac{1}{k^{\alpha}}}\]

Ряд Дирихле расходится при $\alpha \leq 1$ и сходится при $\alpha > 1$.

\[\sum_{k = 1}^{\infty}{\frac{1}{k^{\alpha}}} = \zeta(\alpha)\]

$\zeta(\alpha)$ — дзета-функция Римана. Для чётных $\alpha$ выражается через число $\pi$. Например, $\zeta(2) = \frac{\pi^2}{6}$, а уже для $\alpha = 3$ значение $\zeta(3)$ аналитически неизвестно.
